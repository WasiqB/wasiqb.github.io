When using **coteafs-selenium**, make sure that following tech-stack is available at your end:

## Environments:

Any of the following:

OS Name | Version |
--------|---------|
Mac OSX | Any |
Windows | Any |
Linux   | Any |

## Tools:

Following tools need to be installed before using this framework:

Tools | Version
------|---------
[Java][jdk] | 1.8
Maven | Latest Version
TestNG Plugin for IDE | Latest Version

[jdk]: http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
